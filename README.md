# Lector-TUI-USM
Innovacion Tecnologica 2022-1

Lectur de TUI's de la USM con lector RFID MFRC522 y Raspberry Pi

## Despliegue

- Clonar e instalar SPI-Py
- checkout \<commit version\>

- python3 -m pip install \<librerias\>
- python3 main_tui.py
